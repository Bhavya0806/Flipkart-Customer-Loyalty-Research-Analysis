# Flipkart-Customer-Loyalty-Research-Analysis
Overview

Research project analysing customer loyalty strategies of Flipkart in the rapidly evolving Indian e-commerce market. Focus on identifying factors influencing customer retention and purchasing behaviour.

🎯 Key Areas
Customer Loyalty Programs
Consumer Behaviour Analysis
Survey-Based Research (100 respondents)
Data Analysis & Insights
E-commerce Strategy

🔍 Key Insights
Majority users shop monthly (43%)
Loyalty programs influence spending behaviour
High overall customer satisfaction (80%+ rated 7–10)

💡 Recommendations
Improve personalization using customer data
Enhance loyalty program engagement
Strengthen customer communication
